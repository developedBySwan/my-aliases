# Alias Setup

first - copy the aliases from .aliases in the repo and then

```
   clone this repo
```

open the .bashrc file

```bash
   nano .bashrc
```

add the source path in .bashrc file

```bash
  source ~/.aliases
```

and quit the file and create aliases file

```bash
  touch  .aliases
```

open .aliases file

```bash
  nano  .aliases
```

and then paste the aliases
